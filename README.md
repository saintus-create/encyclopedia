# Encyclopedia of Rhetoric and Composition

A digital encyclopedia featuring 274 entries covering rhetoric from ancient times to the information age. Built with Next.js and Fumadocs.

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Content Structure

Entries are organized alphabetically in the `content/` directory:
- `content/a/` - Entries starting with A
- `content/b/` - Entries starting with B
- etc.

Each letter folder contains individual entry files and an `index.mdx` that lists all entries in that section.
